INTELLARA — Team Portfolio
A responsive team portfolio website built with HTML, CSS, JavaScript, and Bootstrap 5. It showcases the skills, projects, and profiles of four student developers from the INTELLARA team.

✨ Features

Responsive design with glassmorphism cards and smooth animations
Individual profile pages with resume-style layouts
Interactive contact form with real-time validation
Scroll animations using Intersection Observer API
Mobile-first layout, works across all screen sizes
SEO-friendly with semantic HTML and proper heading structure
